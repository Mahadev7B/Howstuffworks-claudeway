# HowStuffWorks — Claudeway 🚀

A kid-friendly explainer app. Ask any *"How does X work?"* question and get a 4-slide illustrated lesson with voiceover — all generated by **Claude** in a single API call.

## Why this version

Your original app used:
- Paid OpenAI image generation (expensive, unpredictable results)
- Server-side audio (storage, latency)
- Multiple API calls per lesson

This rebuild uses:
- ✅ **One** Claude API call generates slide text + SVG illustrations + narration
- ✅ **Zero** image API costs — Claude draws labeled SVG shapes
- ✅ **Zero** audio infrastructure — browser Web Speech API does voiceover
- ✅ Auto-play mode with synced voice-to-slide advance

## Stack

| Layer | Choice |
|---|---|
| Backend | Flask (Python) |
| AI | Claude Sonnet 4.6 via Anthropic API |
| Illustrations | Claude-generated SVG |
| Voiceover | Browser `SpeechSynthesis` API |
| Frontend | Vanilla JS + Jinja2 + CSS |

## Quick start

```bash
# 1. Clone / cd into this folder

# 2. Install
pip install -r requirements.txt

# 3. Configure
cp .env.example .env
# Edit .env and paste your key from https://console.anthropic.com

# 4. Run
python app.py
```

Visit http://localhost:5000

## How it works

1. User types a question like *"How do rockets fly?"*
2. Flask sends it to Claude with a carefully crafted system prompt
3. Claude returns JSON containing 4 slides — each with title, explanation, fun fact, narration text, and a full inline SVG illustration
4. Flask renders the slides in a player UI
5. The browser's built-in speech synthesis reads each slide aloud
6. Auto-play advances to the next slide when voiceover finishes

## Cost per lesson

Roughly **$0.005 – $0.02** per lesson (one Claude Sonnet 4.6 call, ~3k–8k tokens total).

Compare to the previous stack: ~$0.05 – $0.15 per lesson (DALL-E + GPT + TTS).

## Deploying to Render

- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn app:app --bind 0.0.0.0:$PORT --timeout 180`
- Environment variables: `ANTHROPIC_API_KEY`, `ANTHROPIC_MODEL` (optional), `FLASK_SECRET_KEY`
